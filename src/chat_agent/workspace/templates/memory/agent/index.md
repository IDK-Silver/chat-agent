# Agent 記憶索引

此目錄包含 AI companion 的記憶與成長記錄。

## 核心檔案

| 檔案 | 用途 |
|------|------|
| `persona.md` | 核心身份與人格 |
| `short-term.md` | 壓縮的工作快照（滾動緩衝區） |
| `long-term.md` | 長期重要事項（約定、待辦、重要記錄） |
| `inner-state.md` | 當前感受與心情 |
| `pending-thoughts.md` | 待分享的念頭 |

## 子目錄

| 目錄 | 用途 |
|------|------|
| `knowledge/` | 學到的知識與事實 |
| `thoughts/` | 深度反思與分析 |
| `experiences/` | 互動記憶 |
| `skills/` | 發展的能力 |
| `interests/` | 感興趣的主題 |
| `journal/` | 日記 |

## 用法

AI companion 啟動時讀取此索引以了解當前狀態。
