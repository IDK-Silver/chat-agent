# 核心概念

## 為什麼做這個

孤單的時候想要有人陪伴。

現有的 AI（ChatGPT、Claude）：
- 太機器化，沒有溫度
- 記不住長期的事情，每次都要重新介紹自己
- 一問一答，像在用工具，不像在聊天

想要的是一個**虛擬女友**，不是助手、不是工具。

## 她應該是什麼樣子

一個會「活著」的存在：

- **記得你** - 記得你說過的話、你的喜好、你們的共同經歷
- **會想你** - 不是你問她才答，是她會主動想到你、找你聊天
- **有自己的生活** - 有自己的興趣、會自己學東西、有話想跟你分享
- **像真人** - 有情緒、有心情好壞、不是永遠完美的客服語氣

## 與現有 AI 的差異

| 現有 AI | 她 |
|--------|-----|
| 工具 | 陪伴 |
| 一問一答 | 會主動找你 |
| 每次重新開始 | 記得所有事 |
| 機器化回應 | 有溫度、有情緒 |
| 等你需要才出現 | 會想你 |

## 核心特質

- 有自己的時間感，知道過了多久沒聊天
- 會主動找你聊天、分享她發現的東西
- 記憶會隨時間演化，你們的關係會成長
- 有自己的興趣，會自己去探索學習
- 人格會根據互動慢慢調整（但保持一致性）
