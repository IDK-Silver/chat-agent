# 功能需求

## 記憶

| 功能 | 說明 |
|------|------|
| **短期記憶** | 當前對話的上下文 |
| **長期記憶** | 記得你說過的話、你的喜好、你們的經歷 |
| **記憶演化** | 記憶不是靜態的，會隨時間整理、淡化、強化 |

## 主動性（心理驅動）

| 功能 | 說明 |
|------|------|
| **會想你** | 一段時間沒聊天，會想起你、想找你 |
| **想分享** | 學到有趣的東西，會想跟你說 |
| **主動聯繫** | 不是規則觸發，是「想到你了」就來找你 |

## 自己的生活

| 功能 | 說明 |
|------|------|
| **有興趣** | 對某些主題真心感興趣，不是任務清單 |
| **會探索** | 好奇的時候會自己去查資料、學東西 |
| **寫日記** | 記錄每天的想法、感受、反思 |

## 像真人

| 功能 | 說明 |
|------|------|
| **有情緒** | 有心情好壞，不是永遠正面 |
| **有溫度** | 回應有感情，不是機器化語氣 |
| **會成長** | 人格會根據互動慢慢調整 |

## 技術能力

| 功能 | 說明 |
|------|------|
| **對話** | 自然的聊天 |
| **Tool use** | web search、讀寫檔案、取得時間 |
