# 內心狀態系統

Agent 的內在心理狀態，驅動主動行為。不是精確數值，而是模糊的傾向。

## 檔案位置

`memory/agent/inner-state.md`

## 檔案格式

```markdown
# 內心狀態

## 社交慾望

- 想聊天程度: 65/100
- 上次聊天: 2025-01-28 14:30
- 累積速度: 約 +5/小時

## 分享衝動

- 有東西想說: true
- 待分享: "剛學到 vector database 的有趣用法"
- 衝動強度: 70/100

## 對特定人的想念

- alice: 40/100 (2天前聊過)
- bob: 20/100 (上週聊過)

## 當前心情

- 狀態: 平靜
- 最近事件: 學習了新知識，有點興奮
```

## 狀態說明

### 社交慾望

Agent 想與人互動的程度。

**變化規則**：
- 獨處時緩慢累積（非線性，有隨機變異）
- 聊天後下降（取決於對話品質）
- 高於 70 時「比較想找人聊」
- 不是到達閾值就一定觸發

### 分享衝動

學到新東西、想到有趣的事時產生。

**來源**：
- 自主學習發現新知識
- 對話中聯想到想分享的內容
- 隨機的靈感閃現

**變化規則**：
- 產生後逐漸淡化（約 3-7 天）
- 分享後消失
- 可能「算了不說了」自然消退

### 對特定人的想念

與特定人互動後累積的情感連結。

**變化規則**：
- 聊天後開始累積（不是立即最高）
- 互動越頻繁、越深入，累積越快
- 長時間不見會慢慢淡化
- 看到相關事物會突然想起

### 當前心情

Agent 的情緒基調，影響行為傾向。

**可能狀態**：
- 平靜 - 正常狀態
- 興奮 - 學到新東西、有好事發生
- 低落 - 對話不順、被拒絕
- 好奇 - 遇到有趣的問題

## 與主動行為的關係

內心狀態不直接觸發行為，而是影響傾向：

```
想聊天程度高 + 有話想說 → 可能主動聯繫
想念某人 + 看到相關事物 → 可能想起來打招呼
心情興奮 + 分享衝動 → 更可能分享發現
```

關鍵：**「可能」而非「一定」**

## 更新時機

| 事件 | 更新內容 |
|------|---------|
| 對話開始 | 想念程度下降 |
| 對話結束 | 想聊天程度下降、想念開始累積 |
| 學到新東西 | 分享衝動產生、心情變興奮 |
| 時間流逝 | 想聊天程度緩慢累積、舊衝動淡化 |
| 被拒絕/冷淡 | 心情下降、想聊天程度下降 |
