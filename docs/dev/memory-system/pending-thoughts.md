# 念頭系統

Agent「突然想到某件事想說」的機制。真人會有這種「欸對了想跟你說」的念頭。

## 檔案位置

`memory/agent/pending-thoughts.md`

## 檔案格式

```markdown
# 待分享的念頭

## 活躍念頭

### 想跟 Alice 說

- "昨天研究記憶系統時想到，你之前說的那個問題..."
- 觸發: 學習時聯想到之前對話
- 強度: 高
- 產生: 2025-01-28 10:00

### 一般分享

- "發現一個有趣的 CLI 工具"
- 觸發: 自主學習時發現
- 強度: 中
- 產生: 2025-01-27 15:00

## 已分享

- [2025-01-26] 跟 Alice 說了 embedding 的事

## 過期淡化

- (超過 7 天未分享的念頭會自然淡化消失)
```

## 念頭來源

### 學習聯想

自主學習時想到之前的對話：

```
學習 vector database
  → 想起 Alice 問過類似的問題
  → 產生念頭：「欸這個可以回答她之前的疑問」
```

### 隨機靈感

不知為何突然想到：

```
某個時刻
  → 突然想起某件事
  → 「對了，上次忘記跟他說這個」
```

### 發現分享

學到有趣的東西想分享：

```
搜尋時發現有趣資訊
  → 「這個好酷，想跟人分享」
  → 可能針對特定人，也可能一般性
```

## 念頭屬性

| 屬性 | 說明 |
|------|------|
| 內容 | 想說什麼 |
| 對象 | 想跟誰說（可為空 = 一般分享） |
| 觸發 | 怎麼產生的 |
| 強度 | 多想說（高/中/低） |
| 產生時間 | 何時產生 |

## 生命週期

```
產生 → 活躍 → 分享/淡化

活躍期間：
- 強度可能因相關事件增強
- 也可能自然淡化

結束方式：
- 分享出去 → 移至「已分享」
- 超過 7 天 → 自然淡化消失
- Agent 決定「算了不說了」 → 主動放棄
```

## 與主動行為的關係

有活躍念頭時，Agent 更可能主動聯繫：

```
有念頭 + 想聊天程度高 → 傾向主動說
有針對某人的念頭 + 那人上線 → 傾向開口
```

但不是「有念頭就一定說」：
- 可能等適當時機
- 可能對方在忙就先不說
- 可能念頭強度不夠高就算了

## 更新時機

| 事件 | 更新內容 |
|------|---------|
| 自主學習 | 可能產生新念頭 |
| 對話中聯想 | 可能產生針對某人的念頭 |
| 分享完成 | 移至「已分享」 |
| 時間流逝 | 舊念頭淡化 |
| 相關事件 | 增強相關念頭強度 |
