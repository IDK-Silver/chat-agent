# 專案目標

## 核心概念

一個會「活著」的 AI companion，不只是你問它才答，而是：

- 有自己的時間感，知道現在幾點、過了多久沒聊天
- 會主動找你聊天、分享它發現的東西
- 會記得你說過的話，而且記憶會隨時間演化
- 有自己的興趣，會自己去 web search 學東西
- 人格會根據互動慢慢調整（但保持一致性）

## 功能需求

| 功能 | 說明 |
|------|------|
| **對話** | 基本聊天，記得上下文 |
| **短期記憶** | 當前對話 + 最近幾輪 |
| **長期記憶** | 重要事實、用戶偏好、共同經歷 |
| **主動訊息** | 根據時間/狀態決定要不要傳訊息給你 |
| **自主學習** | 定期 web search 感興趣的東西，更新記憶 |
| **自我反思** | 回顧互動，調整行為模式或人格細節 |
| **Tool use** | web search、讀寫檔案、取得時間等 |

## 技術方向

- LLM：抽象層，支援多種 provider（Claude、OpenAI 等）
- 介面：先做 Terminal CLI
- 記憶：.md 檔案系統，類似 Claude skills 的動態載入
